# Advanced Customization

This guide covers advanced customization options for Stylescribe.

## Custom Templates

You can override any default template by creating a matching file in `.stylescribe/templates/`.

## Custom Helpers

Register custom Handlebars helpers for your templates.

## Build Hooks

Use build hooks to run custom code during the build process.
