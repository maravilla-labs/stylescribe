# Quick Start Tutorial

Get up and running with Stylescribe in 5 minutes.

## Step 1: Install Stylescribe

```bash
npm install stylescribe --save-dev
```

## Step 2: Initialize Your Project

```bash
npx stylescribe init
```

## Step 3: Start the Dev Server

```bash
npx stylescribe dev
```

That's it! Your documentation site is now running.
