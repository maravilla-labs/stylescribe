Our component library provides a comprehensive set of reusable UI elements built with accessibility and customization in mind.

## Overview

Components are the building blocks of your interface. Each component is:

- **Token-driven** — Styled using design tokens for consistent theming
- **Accessible** — Built with ARIA attributes and keyboard navigation
- **Responsive** — Adapts to different screen sizes using container queries
- **Customizable** — Override tokens to match your brand

## Categories

Components are organized into functional groups:

| Category | Purpose | Examples |
|----------|---------|----------|
| **Actions** | User interactions | Button, Link, Toggle |
| **Forms** | Data input | Input, Select, Checkbox, Radio |
| **Containment** | Content organization | Card, Container, Layout |
| **Communication** | Feedback & status | Alert, Badge, Spinner |
| **Navigation** | Wayfinding | Tabs, Dropdown, Sidebar |
| **Data Display** | Information presentation | Avatar, Tag, Timeline |

## Usage

Import individual components or use the full bundle:

```html
<!-- Individual component -->
<link rel="stylesheet" href="css/components/button.css">

<!-- Full bundle -->
<link rel="stylesheet" href="css/bundle.all.css">
```
