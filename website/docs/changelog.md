## [0.3.2] - 2026-01-16

### Initial Open Source Release

First public release of Stylescribe under the @maravilla-labs organization.

- Style guide generator for CSS/SCSS design systems
- W3C Design Token support with 46 programmable functions
- Interactive component documentation
- Multi-theme support with dark mode
- Live reload development server
- MCP integration for AI assistants
